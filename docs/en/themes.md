# Theme System

Themes live under:

```bash
themes/<category>/<name>.yaml
```

The skill now uses the full structured theme schema migrated from the reference project.

- `shuimo/default`

## Usage

```bash
--theme macaron/blue
--theme wenyan/purple
--theme shuimo/default
```

## Practical guidance

- **Macaron**: growth, education, lifestyle, mainstream knowledge posts
- **Wenyan**: humanities, essays, literary and cultural topics
- **Shuimo**: calm Chinese-style branding, restrained aesthetics, wellness/seasonal topics
