# WeChat Publish Checklist

Run this before calling `publish_draft.py`.

## Content

- [ ] Topic matches user request
- [ ] Style matches benchmark and target audience
- [ ] No plagiarism from benchmark articles
- [ ] Opening explains value quickly
- [ ] Sections are complete and logically ordered
- [ ] Ending is natural and not over-sold

## Formatting

- [ ] Markdown source saved
- [ ] Title hierarchy clean
- [ ] Lists readable after conversion
- [ ] Paragraphs not too dense
- [ ] Quotes/code blocks, if any, survive conversion acceptably

## Images

- [ ] Cover image exists
- [ ] Cover image dimensions/crop checked
- [ ] Cover image file size checked
- [ ] Body images placed near relevant sections
- [ ] All local images compressed
- [ ] All local images uploaded to WeChat material library

## Metadata

- [ ] Title finalized
- [ ] Author set if needed
- [ ] Digest prepared if needed
- [ ] Source URL/attribution prepared if needed

## Platform risk

- [ ] No obvious medical / finance overclaim
- [ ] No prohibited diversion wording
- [ ] No missing source for borrowed text/images
- [ ] Facts manually spot-checked if AI-generated

## Upload readiness

- [ ] `appid` present
- [ ] `appsecret` present
- [ ] access token obtainable
- [ ] `thumb_media_id` obtainable
- [ ] HTML generated successfully
- [ ] Draft upload payload validated
