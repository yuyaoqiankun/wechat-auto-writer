# WeChat Style Rules

## 1. Article structure

Prefer this structure:

1. Title
2. Lead paragraph: explain the payoff fast
3. 3-6 body sections using `##`
4. Closing summary
5. Natural CTA if needed

## 2. Title rules

- Keep it specific.
- Prefer benefit + scenario + result.
- Avoid extreme clickbait and unsupported claims.
- Avoid overlong titles.

## 3. Opening rules

- Tell readers what they get within the first 2-3 sentences.
- Do not spend too long on empty background.
- Prefer one clear problem statement and one clear promised outcome.

## 4. Body rules

- One section, one main point.
- Keep paragraphs short.
- Lists should be concise.
- Explain terms if the audience may be non-technical.
- Use examples and scenarios instead of abstract slogans.

## 5. Image rules

- Cover image should match topic and tone.
- Body images should appear near relevant sections.
- Avoid too many images; they slow reading.
- Keep visual style consistent.
- Compress before upload.

## 6. Formatting rules for WeChat

- Markdown is source, HTML is delivery.
- WeChat editor handles simple headings, paragraphs, blockquotes, lists, and images best.
- Avoid deeply nested lists and heavy table usage.
- Inline CSS should stay conservative.
- Do not rely on external image URLs for final draft.

## 7. Compliance / platform risks

Flag these before upload:

- medical efficacy claims
- finance/investment guarantees
- extreme or absolute wording
- copyright-sensitive quotes/images without source
- aggressive external diversion or contact bait
- AI-generated facts not manually checked

## 8. Draft-box publishing details

Before upload verify:

- title finalized
- digest available if needed
- thumb image uploaded and `thumb_media_id` ready
-正文 HTML uses WeChat-safe tags
- local images already replaced with WeChat material URLs
- author/source attribution filled where appropriate
